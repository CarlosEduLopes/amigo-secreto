# 🎁 Amigo Secreto

Projeto desenvolvido para praticar lógica de programação, manipulação do DOM e interação com elementos HTML utilizando JavaScript.

---

## 📌 Sobre o projeto

O **Amigo Secreto** é uma aplicação simples que permite cadastrar participantes e realizar o sorteio de forma automática.

Durante o desenvolvimento, foram trabalhados conceitos importantes da programação, como armazenamento de dados em arrays, validações, estruturas de repetição, manipulação da interface e organização da lógica em funções.

Além de ser um ótimo exercício prático, este projeto também serve como peça de portfólio, pois demonstra a criação de uma aplicação funcional e interativa com JavaScript.

---

## 🚀 Funcionalidades

- Adicionar nomes de participantes
- Exibir a lista de participantes na tela
- Validar campos vazios
- Impedir nomes duplicados
- Garantir um número mínimo de participantes para o sorteio
- Realizar o sorteio automaticamente
- Exibir o resultado na interface

## 🎯 Objetivo

Este projeto foi criado para consolidar conceitos fundamentais de lógica de programação e JavaScript, mostrando na prática como desenvolver uma aplicação simples, funcional e interativa.
